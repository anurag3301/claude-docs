# claude-docs

Learning guides produced with Claude

## Contents

### [STM32F411](./stm32f411/)
Deep-dive guides on the STM32F411 microcontroller peripherals: GPIO, UART, SPI, I2C, DMA, ADC, Timers, PWM, EXTI/NVIC, ART Accelerator, Bus architecture, and Power/Reset.

### [Yocto](./yocto/)
Yocto Project / OpenEmbedded guides covering BitBake, devtool, recipes, layers, image recipes, kernel & bootloader, BSP layers, the build system, variables, and the SDK/eSDK.

### [gtf](./gtf/)
Architecture and reference docs for **gtf** (GDB TUI Frontend) — a frontend for GDB built on the GDB/MI protocol, targeting C++23.
